常用的命令

列出现有的镜像

docker images

删除一个镜像

docker rmi <image>

从镜像创建一个容器

端口映射? 

docker run -itP --name <container> <image>

列出所有的容器

docker ps
  • -a: 包含已经停止的

删除一个容器

docker rm <container>

连接到容器

docker attach —detach-keys="ctrl-\\" <container>
